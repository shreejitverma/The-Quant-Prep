## üó∫Ô∏è Learning Roadmap

### [01. Foundations](./01_foundations)
*Prerequisites for entering the field.*
*   **Mathematics:** Calculus, Linear Algebra, Probability & Statistics.
*   **CS Basics:** Data Structures, Algorithms, Python for Finance.
*   **Finance Basics:** Markets, Asset Classes, Financial Modeling.

### [02. Quantitative Data Analysis](./02_quantitative_data_analysis)
*The bedrock of research.*
*   **Time Series:** ARIMA, GARCH, Cointegration, Kalman Filters.
*   **Econometrics:** Hypothesis Testing, Regression Analysis.
*   **Data Sources:** Handling Market Data, Alternative Data.
*   **Exploratory Analysis:** Stock Analysis, Visualizations.

### [03. Financial Engineering & Derivatives](./03_financial_engineering)
*The math behind the markets.*
*   **Derivatives Pricing:** Black-Scholes, Greeks, Volatility Surfaces.
*   **Numerical Methods:** Monte Carlo Simulations, Finite Differences.
*   **Stochastic Calculus:** Brownian Motion, Ito's Lemma.
*   **Fixed Income:** Bond Pricing, Yield Curves.

### [04. Machine Learning & AI in Finance](./04_machine_learning_and_ai)
*Modern predictive modeling.*
*   **Classical ML:** Regression, Classification (SVM, Random Forest) for Markets.
*   **Deep Learning:** LSTM, RNNs for Time Series.
*   **Reinforcement Learning:** Agents for Trading (FinRL).
*   **NLP:** Sentiment Analysis from Financial News.

### [05. Algorithmic Trading](./05_algorithmic_trading)
*Turning theory into profit.*
*   **Strategies:** Statistical Arbitrage, Trend Following, Mean Reversion.
*   **Market Microstructure:** Limit Order Books, Execution, Liquidity.
*   **Backtesting:** Event-driven frameworks, Performance Metrics (Sharpe, Drawdown).
*   **Risk Management:** Portfolio Optimization, VaR.

### [06. Quantitative Development](./06_quantitative_development)
*Building production-grade systems.*
*   **C++ Low Latency:** Memory management, Lock-free structures, Network programming.
*   **System Design:** Trading Engines, Order Management Systems (OMS).
*   **Data Engineering:** ETL Pipelines, Pricing APIs.
*   **Python Production:** Dashboards, Automation.

### [07. Interview Preparation](./07_interview_preparation)
*Cracking the interview.*
*   **Coding Puzzles:** LeetCode-style questions for Quants.
*   **Quant Math:** Brain teasers, Probability puzzles.
*   **Case Studies:** End-to-end problem solving (e.g., Swiss Seba).
*   **Company Insights:** Specific guides for Jane Street, Citadel, etc.

### [08. Research & Resources](./08_research_and_resources)
*Continuous learning.*
*   **Seminal Papers:** Derman, Black-Scholes, Simons.
*   **Datasets:** Sources for historical data.
*   **External Resources:** Books, Courses, Libraries.
